# digital-menu

A digital menu application with admin and customer interfaces.

## Project Structure

- `client/` - React frontend application
- `server/` - Node.js/Express backend API

## Getting Started

### Prerequisites

- Node.js
- MongoDB database

### Installation

1. Install server dependencies:
```bash
cd server
npm install
```

2. Install client dependencies:
```bash
cd client
npm install
```

### Running the Application

1. Start the server:
```bash
cd server
npm start
```

2. Start the client (in a new terminal):
```bash
cd client
npm start
```

## Features

- Customer-facing menu display
- Admin panel for managing menu items
- User authentication
- RESTful API
